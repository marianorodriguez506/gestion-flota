# Gestion de Flota

App movil instalable para reportes inmediatos, plan de trabajo, pedidos, historial e informacion de flota.

## Uso

Se publica como app web en Vercel. En el celular se abre con el link y se puede agregar a la pantalla de inicio.

## Version actual

Esta primera version guarda los datos en el navegador. El siguiente paso es conectar Supabase para usuarios reales y datos compartidos entre mecanicos.
